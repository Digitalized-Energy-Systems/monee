
# Install

## Requirements

- Python >= 3.9
- Recommended: create a fresh virtual environment.

## From source

```bash
git clone <your repo url>
cd monee
pip install -e .[all]
```

## Optional extras

- `gekkopy` / GEKKO for nonlinear optimization.
- `pandas`, `numpy` for data IO.
- For docs: see `requirements-docs.txt`.
